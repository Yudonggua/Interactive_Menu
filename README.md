# Interactive_Menu
# Description
使用QR Code Scanner掃描菜單上的QR Code，透過Listview顯示菜單 ，點擊品項後顯示該品項的3D模型與相關資料。

# Requirement
* Android 7.1
* Sceneform https://github.com/SceneView/sceneform-android.git
